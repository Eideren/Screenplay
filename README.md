## Installation

- Press the `+` sign inside of the Unity Package Manager window
- Click on `Add package from git URL` 
- Paste `https://github.com/Eideren/Screenplay.git` in the text field.
- Press Add

You may have to add a reference to `Unity.TextMeshPro` in `Screenplay/Screenplay.asmdef` through the inspector if you have compilation issues.
